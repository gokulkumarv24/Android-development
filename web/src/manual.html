Android Installation

  1. Download the APK
    Click the download button above or scan the QR code to download EdgeDetectorApp.apk to your Android device.
  2. Enable Unknown Sources
    Go to Settings → Security → Unknown Sources and enable installation from unknown sources.
    Security Note: Only enable this temporarily for installing the app. You can disable it again after installation for security.
  3. Install the App
    Open your file manager, navigate to Downloads, and tap on EdgeDetectorApp.apk to install.
  4. Grant Permissions
    When you first open the app, grant camera permissions and any other requested permissions.
    Tip: The app needs camera access to perform edge detection on live video feed.
  5. Start the Server
    Open the app and tap "Start WebSocket Server" to begin streaming processed video.

Network Setup

Connecting to Web Viewer

  1. Ensure Same Network
    Make sure your Android device and computer are connected to the same Wi-Fi network.
  2. Find Device IP
    In the app, the device IP address will be displayed once the server starts. It will look like:
    192.168.1.100:8765
  3. Access Web Viewer
    Open your web browser and navigate to this IP address to view the processed video stream.
Quick Access: The web viewer is also available at the same address but on port 8082 (e.g., http://192.168.1.100:8082) for direct browser access.

Using the App

  1. Camera Setup
    Point your device camera at objects with clear edges for best edge detection results.
  2. Real-time Processing
    The app processes video in real-time using OpenCV edge detection algorithms.
  3. View Results
    Open the web viewer on any device connected to the same network to see the processed video stream.
  4. Performance Monitoring
    The web interface shows live statistics including FPS, processing time, and connection quality.

Troubleshooting

App won't install
  • Ensure "Unknown Sources" is enabled in Security settings
  • Check if you have enough storage space (at least 50MB free)
  • Try downloading the APK again if the file is corrupted
  • Restart your device and try again
Camera not working
  • Go to Settings → Apps → EdgeDetector → Permissions
  • Enable Camera permission
  • Close other apps that might be using the camera
  • Restart the app
Can't connect to web viewer
  • Ensure both devices are on the same Wi-Fi network
  • Check if firewall is blocking the connection
  • Try using the device's IP address directly
  • Restart the WebSocket server in the app
App crashes or freezes
  • Ensure your device has at least 2GB RAM available
  • Close background apps to free up memory
  • Check if your Android version is 7.0 or higher
  • Clear app cache: Settings → Apps → EdgeDetector → Storage → Clear Cache
