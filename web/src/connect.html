Connection Setup Guide
  1. Install EdgeDetector App
    Download and install the EdgeDetector app on your Android device. Make sure you have granted camera permissions.
    Need the app? Visit the download page to get the APK file and installation instructions.
  2. Connect to Wi-Fi
    Ensure your Android device is connected to the same Wi-Fi network as the computer running this web viewer.
    Current Network: Detecting...
  3. Start WebSocket Server
    Open the EdgeDetector app and tap "Start WebSocket Server". The app will display its IP address and port.
    Important: The server typically runs on port 8765. Make sure no firewall is blocking this port.
  4. Note the IP Address
    The app will show something like "WebSocket Server: 192.168.1.100:8765". Copy this address for manual connection or use the QR code for automatic setup.
    Example: ws://192.168.1.100:8765

  3. Configure Firewall
    If connection fails, ensure your firewall allows incoming connections on port 8081 (web viewer) and port 8765 (Android app).

    Use the QR code above for the easiest connection. The EdgeDetector app can scan this QR code to automatically configure the connection to this web viewer.
  2. Manual Connection
    If QR code doesn't work, manually enter the Android app's WebSocket address in the connection settings on the main viewer page.
    1. Go to main viewer page
    2. Click "Manual Connection"
    3. Enter: ws://[ANDROID_IP]:8765
  3. Verify Connection
    Once connected, you should see live video feed from the Android app's camera with edge detection processing applied in real-time.
    Success Indicators: Green connection status, live FPS counter, and real-time video frames.
  4. Monitor Performance
    The web viewer displays live statistics including FPS, processing time, and connection quality. Use these metrics to optimize performance.

Common Connection Issues

  • Can't see the Android app: Make sure both devices are on the same Wi-Fi network
  • Connection refused: Check if Windows Firewall is blocking the ports (8081, 8765)
  • QR code not working: Try manual connection with the IP address shown in the app
  • No video feed: Ensure camera permissions are granted to the Android app
  • Poor performance: Close other apps on Android device to free up resources
